```text
ML Tuner Python 端采集指标的 API 路径确认：

  ① 算子级指标（l2/l3/自定义latency）：
     GET /jobs/{jid}/vertices/{vid}/subtasks/metrics?get=算子前缀.指标名

  ② 端到端延迟（LatencyMarker）：
     GET /jobs/{jid}/metrics?get=latency.source_id.{src_vid}.operator_id.{sink_vid}...latency_p99

  ③ 内置吞吐/背压指标：
     GET /jobs/{jid} → vertices[].metrics (read-records, backpressured-time 等)

  ④ Checkpoint 统计：
     GET /jobs/{jid}/checkpoints → summary.end_to_end_duration

```
