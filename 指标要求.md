### 预期效果指标要求（Markdown 表格形式）

| 指标类别         | 具体要求                                      | 验证方式与目标说明                                                                 |
|------------------|-----------------------------------------------|------------------------------------------------------------------------------------|
| **端到端时延**  | ≤ 5 秒（采集+处理+传输）                     | Flink Web UI / Prometheus 追踪 P99 延迟；ML-Tuner 硬约束 latency_p99 ≤ 5000 ms     |
| **资源调度**    | 响应时间 ≤ 10 秒，精度误差 ≤ 15%             | DS2 控制器单步收敛；Flink Reactive 模式 + Savepoint 验证误差                       |
| **缓存优化**    | 命中率较 LRU 提升 ≥ 20%；网络传输延迟降低 ≥ 20% | 自定义 Gauge（l2_hit_rate、l3_hit_rate）；RocketMQ+Flink 集成减少冗余传输         |
| **参数调优**    | ML 模型实现自适应调整（并行度、Checkpoint 等）| Bayesian 优化（GP+EI），目标函数 J 最小化；仓库 ML_Tuner 已实现 16 维特征工程     |
| **整体平台**    | 高吞吐（≥2000 条/秒）、Exactly-Once 语义、高可靠 | Flink 1.17.2 + RocketMQ 5.3.0；集群 12 Slots 验证吞吐与 Checkpoint 时长 <30 s     |
