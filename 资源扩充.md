# 地震勘探流处理系统 — 虚拟机硬件扩容方案

> **项目名称**：基于 Flink + RocketMQ 的地震勘探数据实时流处理系统  
> **编写日期**：2025-03-15  
> **版本**：v1.0  
> **目标**：扩充虚拟机资源，为后续压力测试与性能评估奠定硬件基础

---

## 一、现状分析

### 1.1 宿主机硬件

| 项目 | 规格 |
|------|------|
| CPU | AMD Ryzen 9 5900HX — 8 核 16 线程 |
| 物理内存 | 32,160 MB（约 32 GB） |
| 虚拟化平台 | Oracle VirtualBox |
| 宿主机 OS | Windows 10/11 |

### 1.2 当前虚拟机配置

| 节点 | IP | CPU | 内存 | 运行服务 |
|------|------|-----|------|---------|
| node01 | 192.168.56.151 | 4 核 | 9.6 GB | JM + TM + NameServer + Broker(n0) |
| node02 | 192.168.56.152 | 2 核 | 3.7 GB | TM + Broker(n1) |
| node03 | 192.168.56.153 | 2 核 | 3.7 GB | TM + Broker(n2) |
| **合计** | — | **8 核** | **17.0 GB** | — |

### 1.3 现存瓶颈

| 编号 | 问题 | 根因 | 表现 |
|------|------|------|------|
| P1 | Broker 频繁触发流控 | node02/03 仅 3.7GB，Broker + TM 共存内存不足 | `system busy, start flow control` |
| P2 | Flink 算子反压 | TM 堆内存小，Slots 不足 | 2000 节点数据处理延迟高 |
| P3 | DLedger 同步延迟 | Broker JVM 堆过小，PageCache 与应用竞争内存 | Raft 副本同步超时 |
| P4 | 弹性调度空间有限 | 总 Slots = 12，DS2 扩缩容余量不足 | 无法验证动态并行度调整效果 |

## 二、扩容方案设计

### 2.1 资源分配总表

| 节点 | CPU（当前→目标） | 内存（当前→目标） | 运行服务 |
|------|-----------------|------------------|---------|
| node01 | 4 核 → **6 核** | 9.6 GB → **12 GB** | JM + TM + NameServer + Broker(n0) |
| node02 | 2 核 → **4 核** | 3.7 GB → **8 GB** | TM + Broker(n1) |
| node03 | 2 核 → **4 核** | 3.7 GB → **8 GB** | TM + Broker(n2) |
| **VM 合计** | 8 核 → **14 核** | 17 GB → **28 GB** | — |
| 宿主机预留 | — → **2 核** | — → **4 GB** | Windows + Producer + DS2 Python |

> **CPU 超分比**：14 虚拟核 / 16 物理线程 = 87.5%，在可接受范围内。  
> **内存分配比**：28 GB / 32 GB = 87.5%，宿主机预留 4 GB 足够运行 Windows + 轻量级 Java Producer 进程。

### 2.3 扩容前后关键指标对比

| 指标              | 扩容前     | 扩容后    | 提升幅度       |
| --------------- | ------- | ------ | ---------- |
| 总 vCPU          | 8 核     | 14 核   | **+75%**   |
| 总内存             | 17 GB   | 28 GB  | **+65%**   |
| Flink 总 Slots   | 12      | 14     | **+16.7%** |
| TM 总可用内存        | ~6 GB   | 14 GB  | **+133%**  |
| Broker 总 JVM 堆  | ~1.5 GB | 4 GB   | **+167%**  |
| OS PageCache 总量 | ~3 GB   | 7.5 GB | **+150%**  |
